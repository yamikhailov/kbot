# kbot
A simple telegram bot that is accessable at t.me/yamikhailov_bot

## Build
To build the bot use the following command:
```bash
bash build.sh
```

## Usage

Right now the capabilites of the bot are limited, it can display the version number and provide a warm, herzlich greeting:)

```bash
./kbot start
```
You can now interact with the bot through telegram chat
